How to use

  1. Export Synchro reports as .txt for all scenarios and time periods.
  2. Name files like 22700.00 EX AM.txt, 22700.00 NB PM.txt, 22700.00 BD SAT.txt, etc.
  3. Drop all files at once (or select multiple files).
  4. Click Generate Excel File.
  5. Excel will have tabs for AM, PM, and SAT with Existing / No‑Build / Build / Build MIT columns.
Supports TWSC (unsignalized) and Signalized intersections · Synchro 6–12
